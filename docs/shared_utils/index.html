Shared utility library for Soroban smart contracts

This library provides common functions, helpers, and patterns used across all CommitLabs contracts including:

  • Math utilities (safe math, percentages)
  • Time utilities (timestamps, durations)
  • Validation utilities
  • Storage helpers
  • Error helpers
  • Access control patterns
  • Event emission patterns
  • Rate limiting helpers
